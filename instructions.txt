# Как использовать контекст между сессиями

## Быстрый старт
При открытии новой сессии Claude Code отправь первым сообщением:

  Прочитай файл CLAUDE.md в корне проекта и продолжай работу с учётом контекста.

Claude Code автоматически читает CLAUDE.md при старте сессии,
но явное указание гарантирует, что контекст будет учтён полностью.

## Что содержит CLAUDE.md
- Описание проекта и технологий
- Архитектурные правила и структуру папок
- Дизайн-систему (цвета, шрифты, отступы)
- Реестр созданных компонентов
- Лог ключевых решений
- Текущий статус задач

## Как это работает
CLAUDE.md — стандартный файл конфигурации Claude Code.
Он автоматически загружается в контекст при каждой сессии.
Я буду обновлять его по мере развития проекта:
- Добавлять новые компоненты в реестр
- Фиксировать дизайн-токены из Figma
- Обновлять статус задач
- Записывать архитектурные решения

## Если CLAUDE.md не подхватывается автоматически
Отправь сообщение:

  @CLAUDE.md — ознакомься с контекстом проекта и правилами работы.

Или просто:

  Прочитай CLAUDE.md
